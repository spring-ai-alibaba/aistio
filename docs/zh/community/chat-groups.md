交流群用于讨论 AgentScope Go 的使用问题、文档反馈、示例补充和贡献协作。公开入口建立后，会在本页补充 Slack 邀请链接、钉钉群号和二维码。

## 交流建议

- 提问时说明使用的 AgentScope Go 版本、Go 版本和运行环境。
- 反馈问题时附上最小复现代码、错误日志和期望结果。
- 讨论 API 或架构改动时，说明使用场景、替代方案和兼容性影响。
- 涉及凭据、密钥、内部代码或业务数据时，请先脱敏。
