# 行为准则

English version: [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md).

## 我们的承诺

我们承诺为所有参与 Aistio 的人提供尊重、友好且免受骚扰的协作环境。此承诺不因年龄、体型、残障、族裔、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教、性取向或其他个人身份而改变。

## 行为标准

有助于形成积极环境的行为包括：

- 使用友好、包容的语言。
- 尊重不同观点和经验。
- 给予并接受建设性反馈。
- 关注项目和用户的最佳利益。
- 对其他社区成员保持同理心。

不可接受的行为包括：

- 使用性化语言或图像。
- 侮辱、贬损、人身攻击或政治攻击。
- 公开或私下骚扰他人。
- 未经明确许可发布他人的私人信息。
- 其他在专业协作环境中可被合理视为不适当的行为。

## 维护者职责

项目维护者负责解释本行为准则，并可移除、编辑或拒绝不符合本准则的评论、commit、代码、issue、pull request 和其他贡献。

对于不适当、威胁性、冒犯性或有害的行为，维护者可以临时或永久限制相关贡献者参与项目空间。

## 适用范围

本行为准则适用于所有项目空间，包括 GitHub issue、pull request、discussion、文档、示例，以及与本项目相关的公开或私下沟通。
